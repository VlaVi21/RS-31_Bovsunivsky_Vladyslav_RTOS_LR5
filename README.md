### 👨🏼‍💻Функціонал:

**ADC Acquisition (Interrupt Mode):** Зчитування аналогового сигналу з піна PA2 (Channel 2) за допомогою переривань (HAL_ADC_Start_IT). Це дозволяє розвантажити центральний процесор для інших завдань.

**Voltage Conversion:** Отримане 12-бітне цифрове значення (0–4095) конвертується у фізичну величину напруги за формулою:

$$V = \frac{ADC_{value} \times 3.3}{4096}$$

**Threshold Control:** Програмна реалізація компаратора. При перевищенні порогу 0.5 В спрацьовує логіка керування зовнішнім світлодіодом.

**Real-time Processing:** Обробка даних відбувається у callback-функції HAL_ADC_ConvCpltCallback одразу після завершення циклу вимірювання.

---
### 🔵Підключення:

**1. Потенціометр (Вхід сигналу):**

- VCC — до піна 3.3V (Black Pill)

- GND — до піна GND

- Output (середній вивід) — до піна PA2

**2. Світлодіод (Індикація):**

- Cathode (-) — до GND

- Anode (+) — через резистор (~330 Ом) до піна PD13 (або вбудований PC13 для тестування).

---
### 🛠️ Налаштування периферії
Для цієї роботи я обрав реалізацію через переривання (Interrupts). Це дозволяє виконувати аналого-цифрове перетворення у "фоновому" режимі без блокування процесора. Поки АЦП працює, ядро МК виконує основний код, а після завершення вимірювання спрацьовує сигнал переривання, і процесор швидко перемикається в контекст ISR для обробки результату.
Такий підхід є оптимальним для систем реального часу, оскільки дозволяє легко додавати нові функції до проєкту без втрати стабільності.

**Порівняння з іншими методами:**

- Polling (Опитування): Простий метод із затримками, який повністю завантажує процесор очікуванням.

- Interrupts (Мій вибір): Ефективна робота без блокування ядра, ідеально для одиничних або періодичних вимірювань.

- DMA (Direct Memory Access): Передача даних в обхід ядра безпосередньо в пам'ять. Найшвидший метод, що мінімізує навантаження на CPU при потоковій обробці сигналів.

**Що це дає проєкту:**
- Використання переривань гарантує, що керування світлодіодом (реакція на зміну напруги) відбуватиметься миттєво, не заважаючи іншим процесам у системі.
